      • 集团外部个人数据保护政策

        简介
        我们认真看待对您个人数据的保护。新加坡华侨银行很荣幸获颁新加坡个人数据保护信誉标志及APEC跨境隐私规则的认证。这些认证显示我们致力于实施高标准的个人数据保护。新加坡华侨银行也荣获新加坡网络安全局颁发的网络安全信誉标志，此标志是彰显我们所采取的强大安全措施。

        我们希望您能了解我们如何管理您的个人数据，这对我们而言至关重要。这意味着我们必须清楚说明我们如何收集、使用和分享您的个人数据。您向我们提供您的个人数据是自愿的。如果您不向我们提供您的个人数据，我们可能无法向您提供某些产品和服务。我们不会出售您的个人数据。

        我们制定了数据保护管理计划来保护您的个人数据。这包括:

        • 实施有力的安全措施
        • 审查我的业务对个人数据的影响
        • 培训我们的员工
        • 检查我们的供应商以确保他们保护您的个人数据
        • 协助您了解您在个人数据保护方面的权利
        • 解答您对有关我们使用您的个人数据的问题
        • 让您能控制营销信息

        什么是个人数据？
        个人数据是指能识别您个人身份的任何信息，或在与其他资料结合后能识别您个人身份的信息。例如:

        • 您的姓名
        • 您的联系资料
        • 您的财务资讯

        收集您的个人数据
        我们只有在具有正当理由的情况下才会收集您的个人数据。以下是我们会收集您个人数据的例子。当您：

        • 使用我们的服务
        • 与我们的工作人员交谈
        • 参加我们的活动

        使用您的个人数据
        我们会使用您的个人数据的理由包括:

        • 确认您的身份
        • 开发和改进我们的产品和服务
        • 就我们的服务或产品与您联系
        • 保护您和我们本身

        分享您的个人数据
        我们只与可信任的人员和组织分享您的个人数据。如有必要，我们可能会在海外分享您的个人数据。我们在这么做时，会确保个人数据保持安全。

        我们可能与之分享您的个人数据的例子如下:

        • 华侨银行集团属下公司
        • 华侨银行集团属下公司的合作伙伴和供应商
        • 政府机构

        我们对您的承诺
        我们有责任保护您的个人数据。我们将在必要时更新本政策，以反映我们的个人数据管理工作。若有任何更改，我们将随时通知您。 根据您与我们的关系，可能有一项或多项个人数据保护政策适用。有关详情请参阅以下的完整个人数据保护政策。如有任何疑问，请联系我们的个人数据保护办公室：dpo@ocbc.com。

        联系我们
        若要就您的个人数据、我们的个人数据保护政策、撤销您的同意或获取和更正您的个人数据等事宜联系新加坡华侨银行个人数据保护办公室，请使用以下联系资讯：

        • 电子邮件: dpo@ocbc.com
        • 电话: 1800 363 3333

        您可以撤销对收集、使用和披露您的个人数据的同意。如果您撤销同意，我们可能无法继续向您提供产品或服务，管理任何已建立的合同关系，或履行或缔结现有或潜在协议。这可能导致您所签订的任何协议终止，并造成您违反合同义务或承诺。在这种情况下，我们保留我们的法律权利和补救权。

        请注意，如果您的个人数据是由第三方提供给我们，您应直接联系该第三方代表您向我们提出任何询问、反馈、查阅获取和修改要求。

      • 最佳执行政策

        1. 简介

          本文件的目的是详细列出本行营业地的适用法律法规所要求的、就执行某些金融票据的指令时，华侨银行有限公司（“华侨银行”的）最佳执行政策。

        2. 范围

          最佳执行政策适用于华侨银行开展受证券期货法（“SFA”）管制的业务的所有营业地点或其职员根据 SFA 被指定为银行代表的营业地点，但不包括华侨银行的子公司。

          最佳执行政策将列出华侨银行的最佳执行流程的指定原则。如有任何冲突或不一致，以当地法规要求为准。

        3. 治理和管理监督

          本行将持续（至少每年一次）监控最佳执行安排，以确认和执行任何适当的改进。

          华侨银行保留根据其认为必要，不时修改本文件的权力。任何对最佳执行政策的重大修改将在此更新。

        4. 最佳执行的适用性

          本文件适用于本行营业地的管理当局规定的“最佳执行”范围内的所有金融票据。

          新加坡金融管理局未将集合投资计划列入最佳执行范围。这是基于集合投资计划仅在（集合投资计划管理人/经理）的一个地点进行认购和赎回，且仅为一个价格（净资产）。因为就执行地点和/或价格并无酌定权，集合投资计划不在本文件的范围内。

          我们有一个政策和框架用于华侨银行为客户提供最佳的执行结果，但以金融票据为前提，并考虑到客户指令的性质、可供执行金融票据的场所和客户要求我们执行这些指令的优先级别。虽然华侨银行会采取所有必要的步骤来达到最佳执行，华侨银行不保证就每一交易都可以达到最佳的执行结果或确认每一交易都可以全部执行。

        5. 政策要求

          在执行你的指令时，华侨银行将考虑到以下因素：

          1. 价格；
          2. 成本；
          3. 速度；
          4. 执行的可能性；
          5. 交割的可能性；
          6. 指令的大小；
          7. 指令的性质；
          8. 指令的其他方面。

          尽管如此，存在某些这些因素会发生变化的场合，这可能包括但不限于以下：

          1. 指令的类别；
          2. 产品的类别；
          3. 客户提供的特别指示；
          4. 执行指令时的市场的一般性条件。

        6. 特别指示

          如果您就指令或其部分的执行发出了特别指示，华侨银行应确保按照您的指示执行指令。在此情况下，在你的指示有关的部分指令被按照特别指示得到执行时，华侨银行认为已经满足了特别指示。

          如果发出了特别指示，这些指示可能会导致和没有这些指示相比，不那么满意的执行结果。

          如果您就指令的一部分发出了特别指示，华侨银行不认为其解除了未被指示覆盖的那部分指令的最佳执行义务。在此情况下，华侨银行将遵守未被指示覆盖的那部分交易的最佳执行政策。

          如果您发出的特别指示无法被遵守，华侨银行在执行您的指令之前，将告知理由。

        7. 处理指令

          我们有进一步的程序来确保我们即时和公平地执行您的指令，考虑到管理任何您、我们和我们的其他客户之间的潜在利益冲突的必要。

          我们将按顺序即时执行您和其他可比的客户的指令，除非（i）您的指令的特征或当时的市场条件使得这样执行不可行；或（ii）您的利益要求不这样执行。

          如果您未提供特别指示，我们将合理尽快执行和分配指令。

          在执行您的指令时，我们可能决定将您的指令和我们自己的指令或其他客户的指令合并执行。比如，为了使得我们执行更大的交易量，向您提供更好的价格或减少交易成本而合并指令。我们仅会在合并指令总体上不会对您造成不利时合并指令。如果不能执行合并指令的全部，我们将基于公平合理原则进行分配。

          我们将尽力合理尽快的将指令执行和分配结果通知您。

        8. 不受银行控制的事项

          在某些情况下，无论是因为系统错误、市场中断或其他原因，可能需要按照与金融票据中所述的不同方式执行客户的指令。在此情况下，交易单位将会尽力获得当时的市场条件下可能获得的最佳结果。

        9. 有关咨询的联系方式

          如有疑问，请咨询您的客户经理。

      • 分行分析 (Branch Analytics)

        我们的分行和技术团队致力于在您访问我们的分行网点时提升您的体验。方式之一是透过 Branch Analytics，这是一种可带来数据驱动的洞察力的技术，可帮助本行提升您的银行体验。

        这种洞察力可告诉我们配备这种技术的分行中相关活动、产品、推广及广告宣传的开展情况。从这些洞察中，我们可更好地通过推广及广告来策划及改善您的体验。

        但我们深知您在意自身信息的收集、使用及披露方式，我们将在如何保护您的隐私权方面着力思考。我们向您保证 Branch Analytics 将仅以汇总及匿名方式提供数据资料。这意味着我们将按照类别来呈现数据资料，并且不会包括任何个人访客的特定信息。获得的数据不能指向任何个人访客，且仅以汇总、平均值及百分比数值的方式呈现关于分行、产品、推广、活动及广告宣传的表现及成果。例如，我们将会了解访客光顾分行的平均用时。然而，本行不会借此清晰了解相关分行在其访客方面的任何详细个人数据资料，例如其姓名或身份证号码等资料。

        1. Branch Analytics 收集何种数据资料？

          Branch Analytics 旨在了解分行客户的访问方式及华侨银行客户的人口结构信息。此数据资料仅以汇总及匿名方式呈现，且概不得有任何信息指向具体访客。本行亦不会分享或披露任何指向个人访客的任何信息。此类数据信息将以汇总、平均值及百分比数值的方式显示访客的年龄段、性别及访问方式，例如，最常访问分行的访客的年龄段或某一访客光顾分行的平均用时。

        2. Branch Analytics 如何惠及分行访客？

          该技术可让我们持续改善分行访客所享有的服务体验。凭借分行访问方式方面的数据，我们能够改善及调整分行的服务，例如强化分行的布局，改变本行产品及服务的提供方式，增强产品的选择性及可用性，并通过推广及广告来提升产品及服务价值。

        3. 是否会共享访客的个人资料？

          否。我们不会与任何第三方共享您的个人资料作为服务内容之一。

        4. 华侨银行如何储存及保护访客个人资料？

          Branch Analytics 收集的个人资料将按照华侨银行相关数据保护政策予以储存及保护。相关政策详情见 www.ocbc.com/group/policies-zh.

        5. Branch Analytics 如何保护访客的隐私权？

          本行致力于保护访客的隐私及努力构建稳妥的安全措施，保护您的个人数据。本行还制定了综合的数据保护管理方案，以确保本行遵循保护个人数据方面的相关法规义务。更多详情，请参阅华侨银行相关数据保护政策，网址为 www.ocbc.com/group/policies-zh.

        6. Branch Analytics 目前在哪些分行进行推广运用？

          Branch Analytics 目前已在威士马广场 (Wisma Atria) 分行进行推广运用。

        7. 是否能够通过相关分析资料获得关于访客的视频录像或图片资料，或者可指向特定个人？

          否。本行仅以汇总及匿名方式获得关于本行产品、推广及广告宣传方面的数据，以获得关于服务表现方面的洞察。至于 Branch Analytics 抓拍的视频录像，将出于相关安全及分析目的予以保留。

        8. 是否会基于 Branch Analytics 的数据向访客发送定向广告？

          由于我们无法识别任何个人访客，所以不会发送定向广告。

        9. 是否可选择不接受该服务体验？

          由于我们无法识别任何个人访客，所以无法提供退出 Branch Analytics 服务的选项。请注意，Branch Analytics 抓拍的视频录像，将出于相关安全及分析目的予以保留。

      • 网上银行安全

        保护您的信息和资金一直是本行的首要任务。本节向您提供本行的隐私政策，并与您分享有关网上安全的实用信息，同时也会指导您如何防范诈骗。

        网上银行的益处与风险

        1. 华侨银行网上银行为您提供另一种电子渠道，您可以在网上管理您的银行账户并执行银行交易。华侨银行所有客户均可使用网上银行服务。

          通过华侨银行网上银行，您可以查询您的账户余额、转账、支付账单、申购股份、查看您的交易历史，甚至获得即时在线批准住房贷款、信誉信贷和装修贷款。您可以在此处查看本行的网上银行服务。

          通过华侨银行网上银行，您可以随时访问自己的账户，快捷方便，随时随地上网进行交易。您还可以根据自身偏好定制金融投资组合。

        2. 互联网是连接世界各地计算机的广域网络，方便数据传输和交换。由于互联网的开放性，所有基于网络的服务，比如华侨银行网上银行，本质上都面临访问代码／用户 ID／用户名、PIN／密码被盗、病毒攻击、黑客、未经授权之访问和欺诈交易等风险。

          虽然本行已采取必要的安全措施，以防范这些风险，但本行仍无法保证您的交易绝对安全，不受恶意程序的攻击。

          作为网上银行客户，您亦须高度重视保护自身账户信息。

        客户服务

        如果您对使用华侨银行网上银行服务有任何查询、问题、不满、争议或索赔，可致电 1800 363 3333 联络本行客户服务人员。他们全天候为您服务，并将告知您需要立即采取哪些措施和／或本行将会采取的行动。您也可以在 www.ocbc.com 上填写反馈表单后提交，本行在工作时间内收到通知的，将在24小时内回复您，但若次日是星期天或公众假期，则将在下个工作日回复您。

        争议解决

        本行致力于为您提供优质的服务。若您在华侨银行网上银行服务方面发生任何索赔或争议，本行将及时处理。在不损害任何一方立即采取措施向新加坡法院寻求紧急救济或寻求法律补救之权利的情况下，本行将立即对索赔／争议展开调查，并在收到通知后七(7)个工作日内回复您。随后，本行将尽快您与真诚协商，以期迅速、友好地解决此事，使双方都满意。

        网上银行无法使用

        如果华侨银行网上银行服务无法使用，本行建议您通过华侨银行电子大厅、华侨银行电话银行和／或访问本行的任何分支机构进行交易。

        保护您的网上银行访问权限

        华侨银行已采取措施保护您的账户信息。但是，为了确保您的网上安全和账户信息不被泄露，本行建议您遵守以下华侨银行网上银行安全指引：

        1. 您在输入用户名和 PIN 前，应确保您所访问的网站属于华侨银行。这可以通过浏览器中显示的 URL 以及银行数字证书中的名称进行验证。此项预防措施将确保您不会向除华侨银行以外的网站透露您的华侨银行网上银行访问代码和 PIN。
        2. 所有浏览器和应用软件均应升级，支持 SSL 256 位加密或更高的加密标准，并提供最新的安全功能，以确保您获得最高级别的安全保护。
